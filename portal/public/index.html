Deploy an environment

    Environment is live

    A full deploy takes ≈ 45 min — AEM's boot dominates. Stage and prod pause until the reviewer approves in GitHub. Destroy takes ≈ 10 min and removes everything billable.

    1 Credentials

    Held in memory only — you'll re-enter them if the portal restarts. Delete the AWS key in IAM once setup is complete.

    2 AWS account plumbing — once per account

    Creates the OIDC provider, deploy roles, state bucket, seed bucket and budget alarm via the bootstrap-aws workflow. Your AWS key sits in repo secrets only while the run executes, then it's removed automatically.

    3 Licensed AEM binaries — into your private bucket

    You need your own AEM 6.5 license. Files land in your private S3 seed bucket, never in git. Upload files or paste direct-download URLs; you can do them one at a time.

    4 Wire the pipeline variables

    Sets BINARIES_SEED_BUCKET and each environment's AWS_ROLE_ARN automatically. Add an admin password if you plan to deploy with hardening on.
